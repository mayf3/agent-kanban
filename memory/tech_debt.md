## 债务详情

### T001: 技能开发规范缺失

**问题**: 目前缺少统一的技能开发规范，不同技能的目录结构、代码风格、文档格式不一致。

**影响**:
- 代码维护困难
- 新技能开发效率低
- 代码审查标准不统一

**解决方案**:
1. 制定技能开发规范文档
2. 创建技能模板
3. 建立 Code Review 检查清单

### T002: 缺少自动化测试

**问题**: 技能代码没有单元测试和集成测试。

**影响**:
- 修改代码容易引入 bug
- 难以保证代码质量
- 回归成本高

**解决方案**:
1. 选择测试框架
2. 编写核心技能的测试用例
3. 集成到 CI/CD 流程

### T003: 技术文档不完整

**问题**: 部分技能缺少使用文档和架构说明。

**影响**:
- 新人上手困难
- 技术传承困难
- 复用率低

**解决方案**:
1. 补充现有技能文档
2. 建立文档模板
3. 定期文档审查
