# Agent 看板

> Agent 技术路线图与技能管理系统

## 项目简介

Agent 看板是一个用于管理 Agent 技能开发、追踪技术债务、维护技术路线图的看板系统。

## 核心功能

- 📊 **技术路线图** - 维护 Agent 技术发展路线图
- 📋 **技能清单** - 管理所有技能的生命周期
- 🔄 **进度追踪** - 追踪开发中的技能
- 💰 **技术债务** - 记录和优先级排序技术债务

## 项目结构

```
agent-kanban/
├── docs/              # 文档
├── memory/            # 技术看板数据
│   ├── tech_roadmap.md
│   ├── skills_inventory.md
│   ├── tech_debt.md
│   └── best_practices.md
└── scripts/           # 自动化脚本
```
